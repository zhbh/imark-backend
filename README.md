## Features
- Information distribution management
    - Post an event
    - Update an event
    - Delete an event
    - Query events
- Category management
    - Create a category
    - Edit a category
    - Delete a category
    - Search categories
- User management
    - Add a user
    - Change the password
    - Block a user
    - Delete a user
    - Change a role
    - Update a profile
- Favorite management
    - Follow an event
    - Unfollow an event
- Map interaction
    - View all events
    - Locate an event on the map
    - Share an event to social media
    - Favorite an event
- Log in
- Log out
- Sign up

## Getting Started

First, install noed modules:
```bash
npm install
```

Second, run the development server:

```bash
npm run dev
```

Server is running at [http://localhost:3000](http://localhost:3000).

## Other commands

```bash
npm run build //deploy to server
npm run test //test user cases
```
